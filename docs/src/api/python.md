## Installation

Pulsing requires Python 3.10+ and can be installed via pip:

```bash
pip install pulsing
```

For development, clone the repository and install in development mode:

```bash
git clone https://github.com/DeepLink-org/pulsing
cd pulsing
pip install -e .
```
