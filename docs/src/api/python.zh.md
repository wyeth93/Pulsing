# Python API 参考

此页面包含 Pulsing Python 接口的完整自动生成 API 文档。

## 安装

Pulsing 需要 Python 3.10+，可以通过 pip 安装：

```bash
pip install pulsing
```

开发时，克隆仓库并以开发模式安装：

```bash
git clone https://github.com/DeepLink-org/pulsing
cd pulsing
pip install -e .
```
